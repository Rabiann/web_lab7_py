Creating variables

Rust is statically typed language. Type of variable should be inferred in compile type. You can create mutable and immutable variables:


                    fn main() {
                        let x: i32 = 1; // create 32-bit integer
                        let mut y: i128 = 12345; // create mutable variable of 128-bit integer
                        y = 2; you can put new value inside y
                        let z = 2; // type will be inferred (by default is i32 even on 64-bit architecture)
                        let some_str = "my string"; // Honestly, it is not String, it's string slice &str
                    }

Functions


                    fn add(x: i32, y: i32) -> i32 {
                        x + y // in Rust you can omit return statement, simply skip ';' at the end of the line
                    }

Loop


                    fn main() {
                        loop {
                            println!("Hello"); // 'loop' describes infinite loop, just like 'while true'
                        }

                        for i in 0..=10 { // that's a slice representation, just like range(0, 11) in Python
                            println!("{i}"); // prints numbers from 0 to 10 including
                        }
                    }

What is Rust?

Rust is a systems programming language focused on safety, speed, and concurrency. It was created by Mozilla and first released in 2010. Rust is designed to prevent common programming errors, especially memory-related issues like null pointer dereferencing, buffer overflows, and data races in concurrent programs.
